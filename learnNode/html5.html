第二章 表单相关元素和属性
一、原有表单及表单控件
1、表单元素
<from>        用于生成表单，不会产生可视化部分。
   - action    单击"确认"时表单被提交的位置
   - method    提交表单时以何种类型的请求。get(！>2KB)或post(安全)
   - enctype   指定对表单内容进行编码所用的字符集
   - name      指定表单的唯一名字，建议与id值保持一致
   - target    指定以哪种方式打开目标URL(如果请求会打开另一个URL资源)
ectype  指定表单数据的编码方式
   - application/x-www-form-urlencoded: 默认编码方式，只处理表单控件里的value属性值，处理成URL编码方式
   - multipart/form-data: 以二进制处理表单数据，会把文件域或指定文件也封装到请求参数里，上传文件时使用该方式
   - text/plain: 当表单的action属性值为mailto:URL时，使用这种编码方式
<form.../>元表里定义的表单控件的请求规则如下：
   - 每个name对应一个请求参数，没有name就不会生成请求参数
   - 多个表单控件有相同的name属性时，则多个表单控件只生成一个请求参数，只是该参数有多个值
   - 表单控件的name属性指定请求参数名，value属性指定请求数值
   - 如果某表单控件设置了disabled或disable="disable"属性，则该表单控件不再生成请求参数
   - 每个表单控件可以设置tabIndex属性值，让用户无需鼠标用Tab键在各表单上移动

2、input元素
单行文本框：<input type="text" />
密码输入框: <input type="password" />
隐藏域：    <input type="hidden" value=""/>
单选框：    <input type="radio" value="" />
复选框：    <input type="checkbox" value="" />
图像域：    <input type="image" width="" height="" />
文件上传域：<input type="file" />
提交按钮    <input type="submit" value="提交"/>
重设按钮    <input type="reset" value="重置" />
普通按钮    <input type="button" value="按钮一" />
<input.../>元素可以指定id、style、class等核心属性，onclick事件属性，onfocus、onblur等级焦点事件属性，还可以指下以下属性
   - checked    设置单、复选框初始状态是否处于选中状态
   - disabled   设置首次加载时禁用此元素
   - maxlength  该属性是一个数值，设置文本框中所允许输入的最大字符数
   - readonly   指定该文本框内的值不允许用户修改（可以用JavaScript修改）
   - size       该属性是一个数值，指定该元素的宽度
   - src        指定图像域所显示图像的URL,当type="image"时

3、使用label定义标签
<label.../>     对可生成请求参数的表单控件元素进行说明，还可以获得更好的用户体验

4、使用button定义按钮
<button.../>    用于定义一个按按钮，元素内部可以包含普通文本、文本格式化标签、图像等内容
<button>可指定id、class、style等核心属性，还可以指定onclick等级事件属性，还可以指定以下属性：
   - disabled   指定是否禁用此按钮
   - name       指定按钮的唯一名称。应与id属性值保持一致
   - type       指定按钮的类型。只能是button、submit、reset其中之一
   - value      指定该按钮的初始值。可以通过脚本修改

5、列表框和下拉菜单
<select.../>    创建列表框或下拉菜单，与<option.../>元素结合使用，不能指定value属性
<select> 除了可以指定id、style、class等核心属性，仅可以指定onchange事件属性(选项改变触发事件),还可以指下如下属性：
   - disabled   设置禁用该列表框和下拉菜单
   - multiple    设置该列表框或下拉菜单允许多选。=
   - size       设置该列表框内同时显示多少个列表项
//<select id="skill" name="skill" multiple="multiple" size="3" <!--disabled="disabled"-->>
注意：当<select>指定了multiple或size就为列表，否则为下列菜单
<select>只能包含以下两种元素
   - <option.../>   用于定义列表选项或菜单项.可以指定如下属性：
      -- disabled    指定禁用该选项
      -- selected   该选项初始状态是被选中状态
      -- value      指定该选项对应的请求参数
   - <optgroup.../> 用于定义列表英或菜单项的组
      -- label      指定该选项组的标签。这个属性必填。
      -- disabled   禁用该选项组里的所有选项

6、使用textarea定义文本域
<textarea.../>   用于生成多行文本域
<textarea>可以指定onclick等事件属性，还可以指定onselect、onchange两个属性，还可以指定如下属性:
   - cols      指定文本域的宽度，该属性必填
   - rows      指定文本域的高度，该属性必填

  - readAsText(file,encoding)   以文本方式读取文件。wncoding指定字符集，默认值：utf-8

  - onloadend                   FileReader读取数据完成后触发该事件指定的函数。不论成功还是失败
  - onerror                     FileReader读取失败时触发该事件指定的函数

5、使用路径
  在canvas上绘制更复杂的图形，必须在canvas上启用路径，借助于路径来绘制图形
  （1）调用beginPath()方法开始定义路径
  （2）各种方法添加子路径
  （3）closePath()方法关闭路径
  （4）fill()或stroke()方法  来填充或绘制路径边框
   添加子路径方法如下：
  - arc(float x,float y,float radius,float startAngle,endAngle,boolean counterclockwise)
     绘制以(x,y)为圆心，radius为半径，从startAngle角度开始 到endAngle角度结束的圆弧，是否取反路径
  - arcTo(float x,float y,float x2,float y2,float radius)   向当前路径添加一段弧
  - bezierCurveTo(float X1,floatY1,float X2,float Y2,float x,float y) 向当前路径添加一段贝济埃曲线
  - lineTo(float x,float y)     从当前结束点连接到x,y对应点
  - moveTo(float x,float y)     从当前结束点移动到x,y对应点
  - quadraticCurveTo(float cpX,float cpY,float x,float y)   当前路径上添加一段二次曲线
  - rect(float x,float y,float width,float height)          当前路径上添加一个矩形

6、绘制位图

  - void drawImage(Image image,float x,float y,float width,float height)
       把image绘制到x,y处，并缩放宽为width高为height
  - void drawImage(Image image,integer sx,integer sy,integer sw,integer sh,float dx,float dy,float dw,float dh)
       在image上从点sx,sy处挖宽=sw高=sh的图片，放到点dx,dy 并且宽dw高dh

一、使用audio和video元素
  <audio src='demo.ogg" controls>
  <video src="move.webm" controls>
  - src        指定音频、视频的URL地址
  - autoplay   当音频、视频加载完成后自动播放。  指定了会忽略preload属性
  - controls   播放音频、视频时显示控制条
  - loop       音频、视频播放完成后再次重复播放
  - preload    auto:预加载音频、视频 none:不预加载 metadata:只加载音频、视频的元数据
  - poster     指定一张图片URL，当视频下载完成、开始播放前显示该图片
  - width      指定<video>的宽度
  - height     指定<video>的高度

二、使用javascript控制媒本播放
  - play()               播放音频、视频
  - pause()              暂停播放
  - load()               重新装载音频、视频文件
  - canPlayType(type)    判断是否可播放的type类型文件。audio/ogg;audio/mpeg
     返回三种值 probavy:支持  maybe:可能支持 "":不支持
