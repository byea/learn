windows下mysql的安装

一、下载mysql安装包

 1.进入mysql官网 （可以百度）

 2.网页最下面中间点击 MySQL Community Server

 3.点击 第二个Download,下载windows_64位mysql安装包


二、开始安装

1.使用管理员权限进入cmd界面。

2.进入MySQL安装包文件夹bin路径。

3.使用mysqld install命令安装MySQL。
  
4. mysqld --initialize --console

5.使用net start mysql启动mysql服务。

6.使用 mysql -u （root 用户名） -p （1234 密码） 登陆。没有密码，就不写。

7.如果安装不成功需要修改配置文件my.ini,修改MySQL安装路径。
