二、css选择器
1、元素选择器
      E{...}    // a{color:green;}
2、属性选择器
  - E[attr]{...}             对具有attr属性的E元属起作用
  - E[arrt=value]{...}       对具有attr属性且attr值为value的E元素起作用
  - E[attr~=value]{...}      对具有attr属性且attr值为以空格格开的系列值，某个值为value的E元素起作用
  - E[attr|=value]{...}      对具有attr属性且attr值为以连字符格开的系列值，某个值为value的E元素起作用
  - E[attr^="value"]{...}    对具有attr属性且attr值以value开头的字符串的E元素起作用
  - E[attr$="value"]{...}    对具有attr属性且attr值以value结尾的字符串的E元素起作用
  - E[attr*="value"]{...}    对具有attr属性且attr值包含value的字符串的E元素起作用性 
  <--以上选择器匹配越严格优先级就越高-->
3、ID选择器
  #idValue{...}
4、class选择器
  E.classValue{...}    //对类名为classValue的E元素起作用。E可省略
5、包含选择器
  div a{...}   在div内的a标签。   a标签只要在div内部就行
6、子选择器
  div>a{...}   a标签在div内，且是div的子元素。
7、兄弟选择器
  div~.aa      div标签后的类名为aa的兄弟元素
8、选择组合器
  div,a,#abc{...}    //样式对div,a标签和id="abc"的元素都起作用

第二章 字体与文本相关属性
一、字体相关属性
  - font          复合属性:style variant weight size line-height family的复合属性（不建议使用）
  - color               字体颜色
  - font-family         字体  可以多个属性以,隔开，优先找前面的字体
  - font-size           字体大小
  - font-size-adjust    对不同字体尺寸微调
  - font-stretch        改变文字的横向拉伸。有三种值：narrower、normal、wider
  - font-style          设置字体风格，是否斜体等。normal、italic(斜体)、oblique(倾斜字体)
  - font-weight         设置字体是否加粗。lighter、normal、bold、bloder等或用100、200...900
  - text-decoration     控制文字是否有修饰线。有none、blink、underline、line-through、overline
  - font-variant        设置文字的大写字母格式。normal、small-caps(小型的大写字母)
  - text-shadow         设置文字是否有阴影
  - line-height         设置字体的行高
  - letter-spacing      设置字符之间的间隔
  - word-spacing        设置单词之间的间隔

  - color            阴影颜色
  - xoffset          阴影横向的偏移
  - yoffset          阴影纵向的偏移
  - radius           阴影模糊半径
  --text-shadow:color xoffset yoffset length
  --text-shadow:xoffset yoffset radius color

 - 多组阴影使用逗号隔开

  - #000000、#000                 用三位或六位十六进制
  - rgb(12,255,60)                红、绿、蓝
  - hsl(120,100%,100%)            色调、饱和度、亮度
  - rgba(50,122,255,0.3)          红、绿、蓝、透明度
  - hsla(120,100%,100%,0.6)       色调、饱和度、亮度、透明度
三、文本相关属性
  - text-indent        段落文本缩进
  - text-overflow      控制溢出文本的处理方法
   >--clip              如果设置了overflow:hidden，当溢出文本是只裁剪溢出文本
   >--ellipsis          如果设置了overflow:hidden，当溢出文本是裁剪溢出文本并显示...
  - vertical-aglin     目标元素垂直对齐方式
   >--auto              自动对齐
   >--baseline          默认值。文本内容与基线对齐
   >--sub               文本内容与下标对齐
   >--super             文本内容与上标对齐
   >--top               文本内容与元素顶端对齐
   >--middle            文本内容对齐到元素中间
   >--bottom            文本内容与元素底端对齐
   >--length            文本内容相对于基地的偏移距离
  - text-align          文本水平对齐方式。letf、right、center、justify(两端对齐)
  - direction           文本流入的方向。ltr、rtl
  - word-break          文本内容换行方式。
   >--noraml、keep-all  在空格和连字符换行 
   >--break-all         允许单词中间换行  Firefox和Opera可能不支持
  - word-wrap           文本内容换行方式。normal默认、break-word允许单词中间换行

一、背景相关属性

  - background-image       背景图片
  - background-attachment  背景图片是否滚动。scroll默认滚动、fixed固定
  - background-position    背景图片的位置。长度或百分比，只定义一个值就是横坐标，纵50%
  - backgorund-repeat      背景图片是否平铺。no-repeat、repeat-x、repeat-y、repeat

  - background-clip        背景的覆盖范围

  - background-size        背景图片大小。长度、百分比、auto（保持纵横比，宽或高指定一个）

第五章 盒模型与布局相关属性
一、布局相关属性
  - float        控制是否浮动及如何浮动。（display="block")
  - clear        设置组件的左右是否允许出现浮动现象
   >-- none      默认值两边都允许出现浮动
   >-- left      不允许左边出现浮动
   >-- right     不允许右边出现浮动
   >-- both      两边都不允许出现浮动
  - clip         对元素进行裁剪。auto不裁剪，rect(1 2 3 4)在目标上定义一个矩形，目标只有位于该矩形内的区域才会被显示出来
  - overflow     元素区域不够容纳内容时的显示方式。
   >-- visible   不裁剪也不添加滚动条。默认值
   >-- auto      不够容纳时就自动添加滚动条
   >-- hidden    自动裁剪不够显示的内容
   >-- scroll    总是显示滚动条
  - overflow-x   与overflow类似，只控制水平方向的显示方式。内容溢出时出现垂直滚动条
  - overflow-y   与overflow类似，只控制垂直方向的显示方式。内容溢出时出现水平滚动条
  - visibility   对象是否显示。visible/hidden 不显示时也占用空间
  - display      对象如何显示。

  - list-style-position    指定标记出现的位置。outside、inside
  - list-style-type        指定列表标记的样式。decimal、disc、circle、square...

4、media query功能
   CSS增加media query表达式，可以对媒体进行匹配，不同参数的媒进行精确控制
@media not|only 设备类型 [and 设备特性]*
    [and设备特性]可以出现0~N次，通过[and设备特性]可以对多个设备特性进行匹配
  - width               带单位的长度        匹配浏览器的宽度
  - height              带单位的长度        匹配浏览器的高度
  - aspect-ratio        比例值 例:16/9      匹配浏览器的宽、高度比值
  - device-width        带单位的长度值      匹配分辨率的宽度值
  - device-height       带单位的长度值      匹配分辨率的高度值
  - device-aspect-ratio 比例值 例:16/9      匹配分辨率的宽、高度比值
  - color               整数值              匹配设备使用多少位色深
  - color-index         整数值              匹配色彩表中的颜色数
  - monochrome          整数值              匹配单帧缓冲器中每像素位的位数。不是单色设备就为0
  - resolution          分辨率 如:300dpi    匹配设备的物理分辨率
  - scan                progressive或interlace   匹配设备的扫描方式 逐行/隔行
  - grid                0或1                设备是否基于栅格的
   media query语法格式中的设备类型如下：
  - all        适用于所有的设备类型
  - aural      适用于语音和音频合成器
  - braille    适用于触觉反馈设备
  - embossed   适用于凸点字符（盲文）印刷设备
  - handheld   适用于小型或手题设备
  - print      适用于打印机
  - projection 适用于投影图像，如幻灯片
  - screen     适用于计算机显示器
  - tty        适用于使用固定间距字符格的设备，如电传打字机和终端
  - tv         适用于电视类设备
